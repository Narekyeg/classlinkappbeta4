# ClassLink - Ցուցակային Համակարգ

Հայկական դպրոցի ներկայության հսկման համակարգ աշակերտների և ուսուցիչների համար:

## Առանձնահատկություններ

### Աշակերտների համար:
- Գրանցում և մուտք գործելու հնարավորություն
- Ընտրելու հնարավորություն 10, 11, 12-րդ դասարանների միջև
- Յուրաքանչյուր դասարանի համար դասասենյակների ընտրություն (Ա, Բ, Գ, Դ)
- Ներկայության նշում օրական մեկ անգամ (Ներկա/Բացակա)
- Անձնական ներկայության պատմության դիտում

### Ուսուցիչների համար:
- Գրանցում և մուտք գործելու հնարավորություն
- Դասարանի և դասասենյակի ընտրություն
- Աշակերտների ներկայության դիտում ըստ ամսաթվի
- Բոլոր աշակերտների տվյալների և նրանց ներկայության կարգավիճակի դիտում

## Տեխնիկական առանձնահատկություններ:
- Հայերեն լեզվով ամբողջական ինտերֆեյս
- Պատասխանատու դիզայն (Responsive Design)
- Տվյալների պահպանում բրաուզերի localStorage-ում
- Մեկ օրվա համար մեկ անգամ ներկայության նշման հնարավորություն
- Ապահով գրանցում և մուտք գործելու համակարգ
- Մոռացված գաղտնաբառի վերականգնում gmail-ի միջոցով

## Ինչպես օգտագործել:

### Աշակերտների համար:
1. Բացեք index.html ֆայլը բրաուզերում
2. Ընտրեք "Աշակերտ" դերը
3. Գրանցվեք նոր հաշիվ ստեղծելու համար:
   - Մուտքագրեք ձեր անունը
   - Ստեղծեք օգտատիրոջ անուն
   - Սահմանեք գաղտնաբառ
   - Ընտրեք դասարանը (10, 11, կամ 12)
   - Ընտրեք դասասենյակը (Ա, Բ, Գ, Դ)
4. Մուտք գործեք ձեր տվյալներով
5. Նշեք ձեր օրական ներկայությունը (Ներկա/Բացակա)
6. Դիտեք ձեր ներկայության պատմությունը

### Ուսուցիչների համար:
1. Ընտրեք "Ուսուցիչ" դերը
2. Գրանցվեք նոր հաշիվ ստեղծելու համար:
   - Մուտքագրեք ձեր անունը
   - Ստեղծեք օգտատիրոջ անուն
   - Սահմանեք գաղտնաբառ
   - Նշեք ձեր առարկան
3. Մուտք գործեք ձեր տվյալներով
4. Ընտրեք դասարանը և դասասենյակը
5. Ընտրեք ամսաթիվը ներկայությունը դիտելու համար
6. Դիտեք բոլոր աշակերտների ներկայության կարգավիճակը

## Ֆայլերի կառուցվածքը:
- `index.html` - Հիմնական HTML կառուցվածքը
- `styles.css` - Ճկուն և գեղեցիկ ոճաոորում
- `script.js` - Ամբողջ ֆունկցիոնալությունը
- `README.md` - Օգտագործման ուղեցույց

## Նշումներ:
- Տվյալները պահպանվում են բրաուզերի localStorage-ում
- Իրական ծրագրում խորհուրդ է տրվում օգտագործել հարց բազա
- Աշակերտները կարող են ներկայություն նշել օրական միայն մեկ անգամ
- Բոլոր դասարանները ունեն Ա, Բ, Գ, Դ դասասենյակներ
